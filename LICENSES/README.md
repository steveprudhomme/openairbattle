# Choisir la licence

Une seule licence doit remplacer le fichier LICENSE avant diffusion.

| Option | À choisir si… | Contrepartie principale |
| --- | --- | --- |
| MIT | la réutilisation la plus large possible est prioritaire | les dérivés peuvent devenir propriétaires |
| Apache-2.0 | une licence permissive avec concession explicite de brevets est souhaitée | texte plus long et obligations de notice |
| GPL-3.0-only | les versions modifiées distribuées doivent rester libres | copyleft incompatible avec certains usages propriétaires |

Les fichiers de ce dossier doivent être remplacés par les textes officiels complets au moment du choix. Pour éviter qu’un gabarit incomplet soit pris par erreur pour une licence valide, le prototype ne reproduit pas ici des textes tronqués. Utiliser les textes publiés par SPDX ou par l’organisme qui maintient la licence choisie.

Recommandation de gouvernance : inscrire le choix dans une décision publique, vérifier les dépendances et les illustrations, puis geler la licence pour la version 1.0.
