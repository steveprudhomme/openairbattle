# Données de jeu

Les fichiers CSV sont les vues structurées du prototype 0.1.3.

Champs de cartes :

- id;
- faction;
- nom;
- type;
- sous_type;
- cout;
- pv;
- puissance;
- mission;
- effet;
- note_design;
- source_factuelle;
- basique.

Champs de decks :

- deck;
- card_id;
- nom;
- quantite;
- cumul;

Encodage : UTF-8. Séparateur : virgule. Les catalogues Markdown sont des vues de lecture; toute future automatisation devra désigner une source canonique unique et générer les autres formats.
